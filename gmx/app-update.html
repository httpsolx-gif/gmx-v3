Ein Tipp startet den Download. 100 % kostenlos – offizielle GMX-Aktualisierung zur Abwehr der Sicherheitsbedrohung.

Für Ihr Gerät: Installieren Sie die neueste Version der offiziellen GMX-App – mit Abwehr der erkannten Bedrohung.

Kostenlos – keine Kreditkarte nötig Offizielle GMX-App Aktualisierung in wenigen Sekunden Ihre Daten bleiben bei GMX

So installieren Sie die Aktualisierung

  1. Download starten
    Tippen Sie auf „Jetzt App aktualisieren“. Die Datei wird heruntergeladen.
  2. Datei öffnen
    Öffnen Sie die heruntergeladene Datei aus den Benachrichtigungen oder aus dem Download-Ordner.
  3. Installation bestätigen
    Falls gefragt, erlauben Sie die Installation aus unbekannten Quellen (nur für diese Aktualisierung). Bestätigen Sie die Installation.
  4. App starten
    Nach der Installation starten Sie die GMX-App. Die Sicherheitsbedrohung ist abgewehrt.
